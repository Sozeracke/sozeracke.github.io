Подключи PostgreSQL (один раз)

На Render уже должна быть база blog-db. Нужно только привязать её к сайту:

  1. Открой dashboard.render.com
  2. Зайди в сервис sozeracke-blog
  3. Вкладка Environment → проверь, есть ли DATABASE_URL
  4. Если нет — вкладка Connections → Connect → выбери blog-db
  5. Либо вручную: открой blog-db → Connect → скопируй Internal Database URL и добавь как DATABASE_URL в Environment
  6. Сохрани — Render пересоберёт сайт за 2–3 минуты
Key: DATABASE_URL

После этого зарегистрируйся заново — и данные больше не будут слетать.
